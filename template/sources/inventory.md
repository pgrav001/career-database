# Sources Inventory

> _Every raw artifact we know about, and whether it's been mined into the database._

## How to use

- When a new artifact is added (drop in `sources/inputs/` or just discovered on disk), add a row here.
- When an artifact's content has been extracted into the right files (`history/`, `evidence/`, `stories/`, etc.), check the `Mined` box.
- A `Mined: ✓` row doesn't mean "no more value here" — it means "first pass done." Re-mining as the structure evolves is expected.
