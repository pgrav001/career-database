# Themes — deep-dives on each strength

> _One file per signature strength. The expansion layer between `strengths.md` (one-line claim) and `evidence/*.md` (the proof)._

## What goes here

For each of the 6 strengths in `[[strengths]]`, there should be a theme file that:

- **Defines** the strength precisely (what it is, what it isn't)
- **Maps** the chronological evidence that supports it (so you can see the pattern across roles)
- **Captures** anchor quotes from peers / managers / reports that validate it
- **Provides** framing variants — how to position this strength for different audiences (Director loop, Staff IC loop, AI lab, FAANG, startup)
- **Names** the anti-pattern — when does this strength show weakness or backfire?
- **Tracks** open recall — pressure-test thoughts that haven't resolved yet

## Naming

`{kebab-case-strength-name}.md` — examples:
- `ballast-under-fire.md`
- `operating-model-redesigner.md`
- `conviction-before-consensus.md`
- `discipline-shaper.md`

## Why themes exist (vs. just strengths.md + evidence/)

`strengths.md` is the headline layer — one paragraph each, optimized for resume / LinkedIn use.

`evidence/*.md` is the proof layer — atomic, one accomplishment per file.

`themes/*.md` is the **bridge**. It's where you do the deeper thinking about the strength itself — pattern across cycles, the why-it-matters argument, the framing for different audiences. When an interviewer asks "tell me about how you operate," you draw from the theme; when they ask "tell me about a specific example," you draw from the evidence.

## Standard structure (apply to each theme file)

- **# [Strength name]**
- _One-line summary_
- **## Definition** — precise definition
- **## Why this matters in a job search** — the case for surfacing this strength
- **## Evidence — chronological** — the pattern across roles, with `[[evidence/...]]` links
- **## Anchor quotes** — from peers / managers / reports
- **## Interview framing variants** — how to position for different prompts / audiences
- **## When it shows weakness (anti-pattern)** — the risk side
- **## Open recall** — pressure-test thoughts
