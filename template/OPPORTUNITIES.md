# Opportunities — "We could build X"

> _Log of future enhancements to the database, workflow, or artifact set. What we could build, why it'd be valuable, what would trigger building it._

**Why this file exists:** When you say "we could build X" or "we're not building Y yet" in a session, that idea vanishes if it only lives in chat. This file is the durable place for it. Future sessions can review the list and decide what's now worth building.

**Format for each entry:**

```markdown
## Title

**What:** [the idea]
**Why valuable:** [the case for it]
**Trigger:** [what would have to be true for us to build it]
**Status:** open | building | dropped | shipped
**Opened:** YYYY-MM-DD
```
