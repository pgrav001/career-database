# Application Tracker

> _Active and recent applications, with status. The per-application detail lives in `applications/per-application/{slug}.md`._

## Conventions

**Status:**
- `interested` — saved but not applied
- `prep` — preparing resume / cover letter / referral path
- `applied` — submitted
- `screen` — recruiter screen scheduled / in progress
- `loop` — onsite / virtual interview loop
- `offer` — offer received
- `closed-no` — rejected / withdrew
- `closed-yes` — accepted (move details to `[[history/...]]` for the new role)
