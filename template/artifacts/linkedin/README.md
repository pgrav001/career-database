# Generated LinkedIn Copy

LinkedIn About sections, headlines, role descriptions, and post drafts live here. Each is downstream of the substrate.

## Naming convention

`{YYYY-MM-DD}-{purpose}.md` — examples:
- `2026-06-15-about-section.md`
- `2026-06-20-headline-refresh.md`
- `2026-07-01-launch-announcement-post.md`

## Notes

- LinkedIn About sections should lead with the strongest claim, not chronology.
- Headlines have a 220-character limit. Optimize for the recruiter-search keywords without sacrificing voice.
- Recommendations you write for others (and ask others to write for you) — see `[[references]]`.
