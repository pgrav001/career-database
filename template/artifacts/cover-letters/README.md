# Generated Cover Letters

Cover letters generated for specific applications live here.

## Naming convention

`{YYYY-MM-DD}-{company}-{role-shorthand}.md` — examples:
- `2026-06-15-stripe-director-dx.md`
- `2026-06-22-vercel-staff-eng-leader.md`

## Cover letter shape (when in doubt)

1. **Open with the claim** — pick the strength most relevant to the company's stated challenge
2. **Prove with one specific example** — pull from `evidence/`, link `[[stories/...]]` for the spoken version
3. **Connect to their problem** — explicit reference to what the company is doing that you're a fit for
4. **Close with the ask** — short, specific, no salesy filler

Voice: match `voice/self-voice.md` patterns. Avoid the corporate hero voice.
